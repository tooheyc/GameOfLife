The Game of Life
The rules for the Game of Life were devised by British mathematician John Horton Conway. In this version, the numbers in each cell represent the number of live neighbors next to that cell. With each iteration, the following occurs:
  • Any live cell with fewer than two live neighbors dies, as if caused by under-population.
  • Any live cell with two or three live neighbors lives on to the next generation.
  • Any live cell with more than three live neighbors dies, as if by over-population.
  • Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.
To see the inner workings, set the grid to SHOW. The RESET button resets the default pattern. Changing the SIZE also resets the default pattern, spacing the elements appropriately to the new grid, which changes the outcome. The SPEED can be changed at any time without affecting the pattern. The CLEAR button creates a blank slate. To create your own pattern, click on cells to alter their living status.
